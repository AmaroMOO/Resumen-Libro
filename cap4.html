Capítulo 4: Tipos de datos primitivos

  • Tipos: Boolean, Number, BigInt, String, Null, Undefined
  • Booleans: true, false
  • Numbers: enteros y decimales (NaN, Infinity)
  • Strings: texto entre comillas ('hola', "hola")
  • Null = valor nulo, Undefined = sin valor
  • Conversión:
    • Number()
    • String() o toString()
    • Boolean()
